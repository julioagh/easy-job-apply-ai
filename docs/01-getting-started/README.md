# 📖 Getting Started - Documentación inicial

Esta carpeta contiene los documentos esenciales para comenzar a usar el sistema.

## 📄 Archivos

### ⭐ [GUIA_EJECUCION_COWORK.md](GUIA_EJECUCION_COWORK.md) - **EMPIEZA AQUÍ**

**Propósito:** Guía completa paso a paso para ejecutar el sistema en Cowork

**Contenido:**
- Limitaciones de Cowork/LLMs y cómo trabajar con ellas
- Proceso completo Fase 1 → 2 → 3
- Checklist de ejecución
- Errores comunes y soluciones
- Tips para mejores resultados

**Cuándo usar:** Primera vez usando el sistema o cuando tengas dudas sobre el proceso

---

### [CONFIGURACION_PERSONAL.md](CONFIGURACION_PERSONAL.md)

**Propósito:** Configuración de datos personales y personalización del sistema

**Contenido:**
- Cómo actualizar tu CV
- Configurar expectativas salariales
- Personalizar ubicación y modalidad de trabajo
- Ajustar el sistema a tus necesidades

**Cuándo usar:** Después de leer la guía de ejecución, antes de aplicar a tu primera posición

---

## 🚀 Flujo recomendado

1. Lee **GUIA_EJECUCION_COWORK.md** completa
2. Configura tus datos en **CONFIGURACION_PERSONAL.md**
3. Ejecuta `prompts/prompt_fase1_COWORK_DIRECTO.md`
4. ¡Aplica a posiciones!
