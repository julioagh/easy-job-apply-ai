# Research & Analysis Directory

This directory contains strategic analyses, market research, and phase analyses
for job applications and career development.

## Contents

- **phase-analyses/**: Detailed FASE 1, FASE 2 analyses for specific job positions
- **salary-research/**: Market salary analysis and negotiation strategies
- **job-assessments/**: Strategic fit and match analysis for opportunities

These files are version-controlled for reference and improvement over time.
